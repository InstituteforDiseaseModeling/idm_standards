*[Bayesian optimization]: A sequential hypothesis testing strategy to optimize hyperparameter values. 

*[calibration curve]: Plots comparing the observed values against the predicted values. 

*[deterministic]: Characterized by the output being fully determined by the parameter values and the initial conditions. Given the same inputs, a deterministic model will always produce the same output.

*[diagnostic plot]: Visual tools used to assess the validity of statistical model assumptions, particularly for linear regression.

*[history matching]: A statistical method for calibrating models to match historical reference data to more accurately estimate unknown parameter values.

*[HTML]: Hyper Text Markup Language

*[likelihood function]: A measure of how well a statistical model explains observed data by calculating the probability of seeing that data under different parameter values of the model.

*[LMIC]: Low- and middle- income country. 

*[ODE]: ordinary differential equation

*[ODEs]: ordinary differential equations

*[ordinary differential equation]: A differential equation containing one or more functions of one independent variable and its derivatives.

*[parameter sensitivity plot]: Visualization of how changes to input parameters affect the model's output to identify which parameters have the greatest impact.

*[posterior distribution]: The updated probability distribution of an unknown parameter after accounting for observed data, calculated using Bayes' theorem.

*[pseudo-likelihood function]: An approximation to the joint probability distribution of a collection of random variables.

*[reproductive number]: In a fully susceptible population, the basic reproductive number is the number of secondary infections generated by the first infectious individual over the course of the infectious period.

*[SEIR model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following four states: susceptible, exposed, infectious, and recovered.

*[SEIRS model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following five states: susceptible, exposed, infectious, recovered, and susceptible.

*[SI model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following two states: susceptible and infectious.

*[SIR model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following three states: susceptible, infectious, and recovered.

*[SIRS model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following four states: susceptible, infectious, recovered, and susceptible.

*[SIS model]: A generic epidemiological model that provides a simplified means of describing the transmission of an infectious disease through individuals where those individuals can pass through the following three states: susceptible, infectious, and susceptible.
